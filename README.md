# Customer_Behavior_Analysis
Data analytics project showcasing customer behavior analysis using python, sql and power Bi.

# 📊 Data Analytics Project

## 📌 Overview

This project demonstrates an end-to-end data analytics workflow, starting from raw data to actionable insights. It includes data loading, exploratory data analysis (EDA), data cleaning, SQL-based analysis, dashboard creation, and final reporting.

The goal is to showcase practical data analysis skills using industry-standard tools and present insights in a clear, business-friendly manner.

---

## 📂 Dataset

* The dataset contains structured data relevant to the analysis (e.g., sales, customers, operations, etc.).
* It is loaded and processed using Python.
* Data quality issues such as missing values, duplicates, and inconsistencies are handled during preprocessing.

---

## 🛠️ Tools & Technologies

* **Python** (Pandas, NumPy, Matplotlib, Seaborn) – Data processing & EDA
* **SQL** (PostgreSQL / MySQL / SQL Server) – Data querying & analysis
* **Power BI** – Dashboard creation & visualization
* **Gamma** – Presentation (PPT) creation
* **Jupyter Notebook** – Development environment

---

## 🔄 Project Workflow

### 1. Data Loading

* Imported dataset using Python
* Inspected structure, data types, and initial records

### 2. Exploratory Data Analysis (EDA)

* Analyzed distributions and relationships
* Identified trends, patterns, and anomalies
* Visualized data using charts and graphs

### 3. Data Cleaning

* Handled missing values
* Removed duplicates
* Standardized formats and corrected inconsistencies

### 4. SQL Analysis

* Loaded cleaned data into a relational database
* Executed SQL queries to extract insights
* Performed aggregations, joins, and filtering

### 5. Dashboard Creation

* Built an interactive Power BI dashboard
* Included KPIs, charts, and filters for dynamic exploration

### 6. Reporting & Presentation

* Created a detailed report summarizing findings
* Designed a presentation using Gamma for stakeholder communication

## ▶️ How to Run

### 1. Clone the Repository

```bash
git clone <repository-link>
cd <project-folder>
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Run Python Analysis

* Open Jupyter Notebook or Python script
* Execute cells step-by-step for EDA and cleaning

### 4. Setup Database

* Import cleaned dataset into PostgreSQL/MySQL/SQL Server
* Run SQL scripts provided in the `/sql` folder

### 5. Open Dashboard

* Open the Power BI file (`.pbix`) to explore the dashboard

### 6. View Report & Presentation

* Check the `/reports` folder for the analysis report
* Open the Gamma presentation for a summary of insights
